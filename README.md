# Конфигуратор сборки ПК

Это моя учебная консольная программа на Python, которая имитирует магазин комплектующих и позволяет собирать свой компьютер в корзину.

### Что умеет программа:
* Показывать каталог деталей по категориям (видеокарты, процессоры, материнки и т.д.).
* Искать нужную деталь по названию и сразу добавлять её в сборку.
* Считать общую стоимость сборки.
* Сохранять и загружать базу данных магазина из JSON-файла, чтобы ничего не пропадало при перезапуске.

### Как устроен код:
Проект разделен на два основных файла, чтобы код не превращался в кашу:
* `main.py` — отвечает только за вывод текстового меню и общение с пользователем.
* `logic.py` — содержит всю внутреннюю логику: ООП-классы для компонентов, поиск и работу с файлами.

### История проекта (как я его писал):
Проект развивался пошагово, и это видно по истории коммитов:
1. **v1.0** — всё работало на обычных функциях и вложенных словарях.
2. **v2.0** — переписал проект на ООП, добавил классы и базовое сохранение в JSON.
3. **v3.0** — разделил код на логику и интерфейс, навёл порядок в структуре.

### Как запустить:
Просто запустить файл `main.py`.
